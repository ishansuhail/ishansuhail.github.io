SwipeSaver

Swipesaver is a web application and open source project that allows users to collaborate and rate different meals at RPI dining halls. An initiative created to solve the problem for students deciding whether they would want to use their meal swipe.
Started in May 2023 by Ishan Suhail, now the team size is 10 people

How does it work?
Our web app scrapes information off of RPI's meal website using beautifulsoup4 python library and Django python to cleanly display and organize meals, as well as allow them to be rated. As well as location based dining hall finding for convenience

Features currently being added...
- Highest rated items displayed
- Search options for specific food items
- Option for users to upload images of their meals
- Deployment
